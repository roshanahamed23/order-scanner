Google Sheets Integration

Connect to your Google Sheet via Google Apps Script Web App URL. All scans will sync automatically.

Google Apps Script Setup Guide

  1. Go to Google Sheets and create a new spreadsheet
  2. Add these column headers in Row 1: Order ID | Dispatch Date | Dispatch Time | Status | Cancel Date | Cancel Time | Operator | Notes
  3. Go to Extensions → Apps Script
  4. Paste the Apps Script code from the box below
  5. Click Deploy → New deployment → Web app → Execute as Me → Anyone with access
  6. Copy the Web App URL and paste it above
